1. How to build.
Command "make" will take over everything.
Just: make

2. How to run.
Run: sudo ./motor
Press the "start/stop" button on breadboard, it will start the motor.
The current will goes from pin7 to pin2.
Press the "direction" button on breadboard, it will change the direction.
Press the "start/stop" button on breadboard, it will stop the motor.

3. How to terminate.
Press: Ctrl+C
Trigger a keyboard interrupt and the program will exit.
After that, in the bash type: echo $?, you will see 2.
It indicate that the program exit from a keyboard interrupt.
